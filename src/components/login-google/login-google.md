# LOGIN GOOGLE

![LOGIN GOOGLE](https://i.postimg.cc/0Q6CwjVD/Screenshot-2023-01-19-174027.png)

## Description

An authentication method used for login using Google by providing a pop up where users use there Google

## Configuration

1. **client_id** - Comes from the default value as a string e.g `"uhwuhuwhuh3763672byydgygy"`.

### Functionality

1. Displays a pop up for users to login to their Google accounts, if user has already logged in to there user account, no interaction is needed.
