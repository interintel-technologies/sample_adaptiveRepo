# MESSAGES LIST

![message list](https://i.postimg.cc/bJxXcS9t/Screenshot-from-2023-01-20-20-22-11.png)

## Description

This element is used to show a list of the first four messages with a see all button to checkout the rest of the buttons.

## Configuration

1. The messages are configured in the data-source payload.

## Functionalities

1. Clicking on the see all button to iew all messages.
